# Rappi Dashboard - WoodlandFlow

## Project Overview
A JavaScript-based system for generating chatbots using flowcharts, making chatbot creation accessible to non-technical users.

## Tech Stack
- **Runtime**: Node.js 20
- **Framework**: Express
- **Package Manager**: npm

## Project Structure
- `app.js` — Main entry point; Express web server on port 5000
- `package.json` — Dependencies and scripts
- `public/` — Static assets (if added)

## Running the App
```bash
npm start
```
App runs on `http://0.0.0.0:5000`

## Deployment
- Target: autoscale
- Run command: `node app.js`
